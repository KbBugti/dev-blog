CSS - Media Queries

Added in CSS3, the media queries allow to adapt the presentation of the content to a wide range of devices without changing the content itself.

We can change the media queries according to:

  • Type of media (screen, print, ...)
  • Device Width/Height
  • Device Orientation

Usage examples :

  • @media screen (min-width: 400px) { /*your styles here*/ }
  • @media print { /*your styles for print here*/ }
  • @media screen (min-width: 1024px) and (orientation: landscape) { /*your styles here*/ }

Example (resize the browser window to see the changes)

I will have a different background at each break point

↑↑↑


					/*DEFAULT STYLES (for the desktop or mobile depending on your approach)
					------------------------------------*/
					#box {
						background: #ccc;
					}
					#box::after {
						content: "[ no media query, default styles ]";
					}
		
		
					/* MEDIA QUERIES STYLES
					------------------------------------*/
					/* Breakpoint 1 - landscape tablets  */
					@media (max-width: 1024px) {
						#box {
							background: #AED6F1;
						}
						#box::after {
							content: "@media (max-width: 1024px)";
						}
					}
		
					/* Breakpoint 2 - portrait tablets  */
					@media screen and (max-width: 768px) {
						#box {
							background: #A569BD;
							color: #fff;
						}
						#box::after {
							content: "@media (max-width: 768px)";
						}
					}
		
					/* Breakpoint 3 - landscape phones  */
					@media screen and (max-width: 667px) {
						#box {
							background: #CB4335;
						}
						#box::after {
							content: "@media (max-width: 667px)";
						}

						pre {
							display: none;
						}
					}
		
					/* Breakpoint 4 - portrait phones  */
					@media screen and (max-width: 480px) {
						body {
							text-align: center;
						}

						#box {
							background: #2E4053;
						}
						#box::after {
							content: "@media (max-width: 480px)";
						}
					}
		
					/*ONLY FOR PRINT*/
					@media print {
						body {
							width: 100%;
							margin: 0;
							padding: 0;
							color: #000;
							background: #fff;
						}
						#box {
							background: #fff;
							color: #000;
						}
						/*be sure to start on a new page to avoid to cut out code lines*/
						pre {
							page-break-before: always; 
						}
						@page {
							margin: 2cm;
						}
					}
